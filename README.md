# wanandroid-flutter

[wanandroid.com](https://wanandroid.com) 网站的App版本，感谢该站点提供的免费API。

## 开发环境

- Android Studio 4.2.1
- Dart 2.14.4 (stable)
- Flutter 2.5.3 (stable)

以上是我当前的开发运行环境，仅供参考。
